O HTML é uma linguagem de MARCAÇÃO e é ESTÁTICO
Apesar da maneira como é escrito, o HTML não é uma linguagem de programação propriamente dita. Inclusive, veja que em seu nome (Hyper Text Markup Language) não há a presença da palavra "programming". O HTML foi criado apenas para apresentar informações, isto é, sem dinamismo.
